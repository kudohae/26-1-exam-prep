핵강세 Pitch Distance 측정

이 가이드는 연구팀 전원이 동일한 방식으로 분석을 수행할 수 있도록 PRAAT 세팅부터 스크립트 실행, 결과 해석까지 전 과정을 기술한다.

PRAAT 6.x Filtered Autocorrelation Floor 75 Hz · Ceiling 300 Hz 단위: Semitone 인당 15–20문장
§ 01 — Overview

분석 개요

문장 내 초점 단어(focal accent)의 F0 peak와 동일 문장 내 비초점 내용어들의 F0 peak 평균 사이의 차이를 Pitch Distance (PD)로 정의하고, 이를 semitone 단위로 측정한다.

PD (st) = 12 × log₂ ( F0_focus / F0_nonfocus_mean )
F0_focus: 초점 단어 안정 구간 F0 peak (Hz)
F0_nonfocus_mean: 동일 문장 내 비초점 내용어 F0 peak 평균 (Hz)

TextGrid Tier 구조

Tier이름내용레이블 예시
1sent문장 경계. 분석 단위s01, s02 …
2word내용어 세그먼트만 표시. 기능어는 비워둠teammates, really …
3focus초점 단어만 f 입력. 나머지 비워둠 → 자동으로 nf 처리f (초점만)

내용어 기준

명사·동사·형용사·부사만 word tier에 입력한다. 관사·전치사·조동사·대명사·접속사는 입력하지 않는다 (기능어 제외 시 평균 F0 과대추정 방지).

§ 02 — Preparation

파일 준비 및 오디오 확인

파일 구성

Sound 파일(.wav)과 TextGrid 파일(.TextGrid)을 같은 폴더에 보관한다. 파일명은 통일한다.

예: 파일명.wav / 파일명.TextGrid

오디오 불러오기 및 파형 확인

PRAAT Objects 창 → Open → Read from file... → wav 파일 선택

Objects 창에서 Sound 선택 → View & Edit으로 파형 확인.

확인 항목: ① 샘플링 레이트 44100 Hz 이상 ② 침묵 구간에서 파형이 0에 근접 (소음 적음) ③ 파형이 한계선에 딱 붙어 잘린 구간(클리핑) 없음

§ 03 — TextGrid Labelling

TextGrid 생성 및 레이블링

TextGrid 생성

Objects 창에서 Sound 선택 → Annotate → To TextGrid...

All tier names 입력란에: sent word focus

Which of these are point tiers: 비워둠 → OK

편집 창 열기

Objects 창에서 Sound + TextGrid 두 개만 선택 (Ctrl 클릭) → View & Edit

Pitch curve를 함께 보려면: Editor 상단 → Pitch → Show pitch

레이블링 순서

① sent tier: 분석할 문장 구간을 드래그 선택 후 경계 추가 → s01, s02 … 입력

② word tier: 내용어 구간마다 경계 추가 → 단어 철자 입력. 기능어 구간은 비워둠

③ focus tier: word tier와 동일한 경계로 초점 단어 구간에만 f 입력. 나머지는 비워둠

경계선 수정 및 삭제

경계선 클릭(빨갛게 선택됨) → Editor 상단 → Boundary → Remove

경계선이 잘 눌리지 않으면 해당 구간을 먼저 확대(Ctrl + 마우스 휠)한 후 시도

맨 처음과 맨 끝 경계선은 삭제 불가 (전체 파일 경계)

TextGrid 저장

Objects 창에서 TextGrid 선택 → Save → Save as text file...

Sound 파일과 같은 폴더에 저장. 확장자 .TextGrid 자동 부여

주의 — Focus tier 경계 일치

focus tier의 경계선은 word tier와 반드시 일치해야 한다. 경계가 다르면 스크립트가 엉뚱한 레이블을 읽어온다. 초점 단어가 두 단어인 경우 word tier에서도 두 개로 나누고 focus tier에서도 각각 f를 입력한다.

유용한 단축키

단축키기능
Ctrl + Enter선택 구간에 경계 추가
Tab다음 interval로 이동
Ctrl + STextGrid 저장
Ctrl + 마우스 휠파형 확대/축소
§ 04 — Pitch Settings

Pitch 설정

스크립트가 내부적으로 Pitch 객체를 자동 생성하므로 수동으로 만들 필요는 없다. 설정값은 아래를 기준으로 한다.

항목설정값이유
방식To Pitch (filtered ac)배경 소음과 유성음 판단 오류에 강함. 인터뷰 음원 기본값
Pitch floor75 Hz성인 남성 하한값
Pitch ceiling300 Hz성인 남성 상한값
Time step0.0 (자동)PRAAT 자동 설정

Octave Jump 수동 검수

스크립트 실행 후 결과값이 지나치게 높거나 낮은 단어가 있으면, Editor에서 해당 구간의 Pitch tier(파란 선)를 육안으로 확인한다. 급격히 튀는 구간이 있으면 해당 단어는 분석에서 제외한다.

§ 05 — Script

PRAAT 스크립트

PRAAT 상단 메뉴 → Praat → New Praat script → 아래 스크립트를 붙여넣고 파일명 두 줄만 수정한 후 실행 (Ctrl + R).

PRAAT Script 
# ── 설정 ────────────────────────────────────────────
sound_file$ = "파일명.wav"      # ← 실제 파일명으로 수정
tg_file$    = "파일명.TextGrid"  # ← 실제 파일명으로 수정
pitch_floor   = 75
pitch_ceiling = 300
# ────────────────────────────────────────────────────

# 파일 불러오기
sound = Read from file: sound_file$
tg    = Read from file: tg_file$

# Pitch 생성
selectObject: sound
pitch = To Pitch: 0.0, pitch_floor, pitch_ceiling

# tier 번호 찾기
selectObject: tg
n_tiers = Get number of tiers
sent_tier  = 0
word_tier  = 0
focus_tier = 0
for i to n_tiers
    name$ = Get tier name: i
    if name$ = "sent"
        sent_tier = i
    endif
    if name$ = "word"
        word_tier = i
    endif
    if name$ = "focus"
        focus_tier = i
    endif
endfor

# 단어별 출력 헤더
appendInfoLine: "sent", tab$, "word", tab$, "focus", tab$, "f0_peak_hz", tab$, "f0_peak_st"

# sent tier 순회
selectObject: tg
n_sent = Get number of intervals: sent_tier

for s to n_sent
    selectObject: tg
    sent$ = Get label of interval: sent_tier, s

    if left$(sent$, 1) = "s" and number(mid$(sent$, 2, 10)) <> undefined

        sent_start = Get start time of interval: sent_tier, s
        sent_end   = Get end time of interval: sent_tier, s

        f_peak   = undefined
        nf_sum   = 0
        nf_count = 0

        n_words = Get number of intervals: word_tier

        for i to n_words
            selectObject: tg
            word$ = Get label of interval: word_tier, i

            if word$ <> ""
                t_start = Get start time of interval: word_tier, i
                t_end   = Get end time of interval: word_tier, i
                t_mid   = (t_start + t_end) / 2

                if t_mid >= sent_start and t_mid <= sent_end

                    focus_i = Get interval at time: focus_tier, t_mid
                    focus$  = Get label of interval: focus_tier, focus_i

                    selectObject: pitch
                    f0_hz = Get maximum: t_start, t_end, "Hertz", "Parabolic"

                    if f0_hz <> undefined
                        f0_st = 12 * log2(f0_hz / 100)

                        # 단어별 출력
                        appendInfoLine: sent$, tab$, word$, tab$, focus$, tab$, fixed$(f0_hz, 2), tab$, fixed$(f0_st, 3)

                        if focus$ = "f"
                            f_peak = f0_st
                        elsif focus$ = "" or focus$ = "nf"
                            nf_sum   = nf_sum + f0_st
                            nf_count = nf_count + 1
                        endif
                    endif

                endif
            endif
        endfor

        # 문장별 PD 요약 출력
        if nf_count > 0 and f_peak <> undefined
            nf_mean = nf_sum / nf_count
            pd = f_peak - nf_mean
            appendInfoLine: "  → PD(", sent$, "): f=", fixed$(f_peak, 3), " nf_mean=", fixed$(nf_mean, 3), " PD=", fixed$(pd, 3), " st (nf=", nf_count, ")"
        else
            appendInfoLine: "  → PD(", sent$, "): f 또는 nf 없음"
        endif

        appendInfoLine: ""

    endif
endfor

# 객체 정리
selectObject: sound, tg, pitch
Remove

실행 방법

스크립트 창 열기

PRAAT 상단 → Praat → New Praat script

파일명 수정

스크립트 상단 두 줄의 "파일명.wav"와 "파일명.TextGrid"를 실제 파일명으로 수정한다.

실행

Run → Run 또는 Ctrl + R

결과는 PRAAT Info 창에 출력된다.

데이터 정리

Info 창 결과를 전체 선택 복사 후 Excel 또는 Google Sheets에 붙여넣으면 탭 구분 텍스트로 자동 정렬된다. 화자별로 PD 값만 따로 모아 평균·표준편차를 산출한다.

제외 기준

① f0_peak_hz가 undefined로 출력된 단어 — 해당 구간 무성음 또는 Pitch 추출 실패. 분석 제외.
② 결과값이 주변 단어 대비 지나치게 튀는 경우 (octave jump 의심) — Pitch tier 육안 확인 후 필요 시 제외.
③ 비초점 내용어(nf_count)가 1개 이하인 문장 — PD 평균의 안정성 부족. 제외 권장.

초점 판정 이중 기준 (반드시 준수)

초점 단어(f)는 ① 청취 시 pitch 상승이 주변 단어 대비 두드러지게 지각되고, ② PRAAT Pitch tier에서 해당 단어의 F0 peak가 동일 문장 내 다른 내용어 평균보다 높게 나타나는 경우에만 표시한다. 두 기준 중 하나라도 충족하지 않으면 해당 문장은 분석에서 제외한다.
